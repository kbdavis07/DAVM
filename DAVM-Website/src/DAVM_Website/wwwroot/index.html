DASHBOARD-AZUREVM
OPEN SOURCE AZURE VIRTUAL MACHINES
MANAGEMENT SOFTWARE

The main objective of Dashboard-AzureVM is to simplify the life of System Administrator and have the ability to perform standard tasks to the Virtual Machines on the cloud. It's developed by Gianluca Bertelli and Daniele Maso.

Dashboard-AzureVM allows to monitor and perform administrative tasks on the Virtual Machines created on your Azure Subscription. The ability to apply the desired actions to all VMs, like start and stop, it becomes more crucial in a Infrastructure as a Service (IaaS) environment. It's the typical scenario for test environment where you need to pause all VMs or start a bunch of them for a new demo.

Project Features

AZURE SUBSCRIPTION INTEGRATION
Support multiple subscriptions

List of all Virtual Machines created for each Azure Subscription, easy setup with subscription import wizard!

VIRTUAL MACHINE MANAGEMENT
Start / Stop control

Ability to manage a single instance or all VMs simultaneously. Save your time!

REMOTE CONNECTION
Direct link for RDP/SSH

Just one click to connect remotely, it use RDP for Windows machines and your preferred SSH for *nix. Taskbar integration for quick launch

DETAILED INFO
Get all VM info at glance

Displays the most useful information for each Virtual Machine. Direct link to Microsoft Azure SDP package for a fast troubleshooting
